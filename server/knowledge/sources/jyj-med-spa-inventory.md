# JyJ Med Spa — inventory management system (Jose Correa)

Use this to answer questions about **why** the system was built, **objectives**, **goals**, **purpose**, or **business value**.

## Objectives

- **Operational:** Stop relying on spreadsheets and memory for retail/supply counts; one source of truth staff could use daily without heavy retraining.
- **Visibility:** Real-time counts across 20+ SKUs, search, and low-stock awareness so reordering wasn’t guesswork.
- **Trust & control:** Firebase Authentication so only approved users could edit inventory—fewer accidents, better auditability, owner confidence.
- **Adoption:** Electron desktop app packaged for non-technical staff, with documented workflows so the spa didn’t depend on a developer for every change.

## Summary one-liner

The objective was to give a small, growing med spa **accurate, secure, easy-to-use inventory** instead of manual notes and spreadsheets.
